## 简介

Q2系统通过不同物理通信方式的转换，实现数据在对应的设备端间传递

---

## 组成

#### [M4短距无线](https://github.com/OS-Q/M4)

通过短距无线接入方式（WIFI)将数据接入到互联网

#### [M5长距无线](https://github.com/OS-Q/M5)

通过长距无线通信方式将数据接入到互联网

#### [M6有线路由](https://github.com/OS-Q/M6)

通过有线网络实现相应数据的转换和接入
